# 搜索网站

## 项目描述
### 排版
```
页面排版依靠html、css、element-ui
```

### 功能
```
功能上通过vue.js vue-router ajax 来实现：
在home主页面实现登录账号、切换搜索引擎（mysearch，百度，谷歌）、信息检索的功能
在search次页面也拥有实时信息检索功能，通过关键字与本地数据（json文件）进行匹配
```
